# useClickAway

监听元素外部的点击事件。适用于模态框或下拉菜单。

:::info
`useClickAway` 是 [`useClickOutside`](/zh-Hans/element/useClickOutside) 的别名。它们具有完全相同的功能和 API。
:::

`useClickAway` 检测发生在引用 DOM 元素外部的点击（鼠标和触摸事件），并在发生时调用回调。传入要保护的元素的 ref 和处理函数。你还可以传入 `enabled` 标志来有条件地激活或停用监听器。

### 使用场景

- 当用户点击外部时关闭下拉菜单、弹出框或模态框
- 当焦点移走时取消选择或停用内联编辑组件
- 在外部交互时关闭通知提示或上下文菜单

### 注意事项

- **事件类型**：监听 `mousedown` 和 `touchstart` 事件，覆盖桌面和移动交互。
- **清理**：当组件卸载或 `enabled` 设为 `false` 时，事件监听器会自动移除。
- 此 hook 是 `useClickOutside` 的别名。使用你偏好的名称即可——它们共享相同的实现。

## 用法

```tsx live
function Demo() {
  const [visible, setVisible] = useState(false);

  const modalRef = useRef<HTMLDivElement>(null);

  useClickAway(modalRef, () => {
    setVisible(false);
  });

  return (
    <div>
      <button onClick={() => setVisible(true)}> 打开模态框</button>
      {visible && (
        <div
          ref={modalRef}
          style={{
            position: "fixed",
            left: "50%",
            top: "50%",
            transform: "translate(-50%,-50%)",
            width: 420,
            maxWidth: "100%",
            zIndex: 10,
            border: "1px solid var(--c-input-border)",
            background: "var(--c-bg)",
          }}
        >
          <div
            style={{
              padding: "0.4em 2em",
              boxShadow: "2px 2px 10px var(--c-box-shadow)",
            }}
          >
            <p
              style={{
                fontWeight: 700,
                fontSize: "1.4rem",
                marginBottom: "2rem",
              }}
            >
              演示模态框
            </p>
            <p>点击模态框外部关闭它。</p>
          </div>
        </div>
      )}
    </div>
  );
};

```
