## Not Ported (Intentional)

| Feature | Reason |
|---|---|
| i18n (zh-Hans, zh-Hant) | Large scope; can be added later |

| LLM text auto-generation | Can be added as a build script later |

## Architecture Differences

| Aspect | Docusaurus | Astro |
|---|---|---|
| Framework | React SPA + SSR | Static Site Generation (MPA) |
| Interactive components | Full React hydration | Astro Islands (client:visible) |
| CSS | Infima framework + custom.css | Custom OKLCH design system |
| Code highlighting | Prism (react-live) | Shiki dual-theme + Prism (react-live) |
| Build output | 127 pages | 128 pages (+ 404) |
| Content management | File-based (docs/) | Content Collections with Zod schema |
| Remark processing | Docusaurus pipeline | Custom remark plugins |
| Bundle strategy | Full React bundle | Selective hydration (Islands) |
