### useFps

#### Returns

`number`: 每秒帧数

#### Arguments

| 参数名  | 描述 | 类型                                         | 默认值 |
| ------- | ---- | -------------------------------------------- | ------ |
| options | -    | [UseFpsOptions](#UseFpsOptions) \| undefined | -      |

### UseFpsOptions

| 参数名 | 描述              | 类型   | 默认值 |
| ------ | ----------------- | ------ | ------ |
| every  | 每过 x 帧计算一次 | number | `10`   |
