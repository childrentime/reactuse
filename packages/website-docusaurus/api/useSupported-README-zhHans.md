### useSupported

#### Returns

`boolean`: 浏览器是否支持

#### Arguments

| 参数名   | 描述                           | 类型                     | 默认值  |
| -------- | ------------------------------ | ------------------------ | ------- |
| callback | 测试回调                       | () => unknown **(必填)** | -       |
| sync     | 使用 useLayoutEffect来进行测试 | boolean \| undefined     | `false` |
