### useInterval

#### Returns
`void`

#### Arguments
|参数名|描述|类型|默认值|
|---|---|---|---|
|callback|回调|() => void  **(必填)**|-|
|delay|时间，如果为 `null` 的话则停止计时器|number \| null \| undefined |-|
|options|可选参数|[UseIntervalOptions](#UseIntervalOptions) \| undefined |-|

### UseIntervalOptions

|参数名|描述|类型|默认值|
|---|---|---|---|
|immediate|是否立即执行。|boolean |`-`|
