### useFavicon

#### Returns

`void`

#### Arguments

| 参数名  | 描述                      | 类型                | 默认值 |
| ------- | ------------------------- | ------------------- | ------ |
| href    | 图标路径                  | string **(必填)**   | -      |
| baseUrl | 基础 url                  | string \| undefined | -      |
| rel     | 设置 link 标签的 rel 属性 | string \| undefined | `icon` |
