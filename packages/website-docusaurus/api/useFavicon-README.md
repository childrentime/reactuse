### useFavicon

#### Returns

`void`

#### Arguments

| Argument | Description                       | Type                  | DefaultValue |
| -------- | --------------------------------- | --------------------- | ------------ |
| href     | icon href                         | string **(Required)** | -            |
| baseUrl  | base url                          | string \| undefined   | -            |
| rel      | set rel attribute to link element | string \| undefined   | `icon`       |
