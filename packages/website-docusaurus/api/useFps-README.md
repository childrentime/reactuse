### useFps

#### Returns

`number`: frames per second

#### Arguments

| Argument | Description | Type                                         | DefaultValue |
| -------- | ----------- | -------------------------------------------- | ------------ |
| options  | -           | [UseFpsOptions](#UseFpsOptions) \| undefined | -            |

### UseFpsOptions

| Property | Description                          | Type   | DefaultValue |
| -------- | ------------------------------------ | ------ | ------------ |
| every    | Calculate the FPS on every x frames. | number | `10`         |
