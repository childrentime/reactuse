### useElementSize

#### Returns
`readonly [number, number]`: 包含以下元素的元组：
- 元素宽度。
- 元素高度。

#### Arguments
|Argument|Description|Type|DefaultValue|
|---|---|---|---|
|target|dom element|React.RefObject&lt;Element&gt;  **(Required)**|-|
|options|options passed to `resizeObserver`|ResizeObserverOptions |-|
