# useInterval

计时器。你可以通过传递 `controls` 参数来手动控制计时器。

## Usage

```tsx live

function Demo() {
  const [count, setCount] = useState(0);

  useInterval(() => {
    setCount(count + 1);
  }, 1000);

  return <div>计数: {count}</div>;
};

```
