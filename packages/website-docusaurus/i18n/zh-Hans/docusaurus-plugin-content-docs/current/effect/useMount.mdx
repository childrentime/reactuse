# useMount

组件挂载后执行一次副作用的 React Hook

## Usage

```tsx live

function Demo() {
  const [value, setValue] = useState("未挂载");
  useMount(() => {
    setValue("已挂载");
  });
  return <div>{value}</div>;
};

```
