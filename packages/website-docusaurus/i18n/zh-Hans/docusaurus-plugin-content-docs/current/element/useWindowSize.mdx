# useWindowSize

跟踪视窗大小的 React Hook。

## Usage

```tsx live

function Demo() {
  const { width, height } = useWindowSize();

  return (
    <div>
      <p>
        width: {width}, height: {height}
      </p>
    </div>
  );
};

```
