# useWindowSize

跟踪视窗大小的 React Hook。

## Usage

```tsx live

function Demo() {
  const { width, height } = useWindowSize();

  return (
    <div>
      <p>
        宽度: {width}, 高度: {height}
      </p>
    </div>
  );
};

```
