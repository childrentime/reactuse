# useWindowsFocus

使用 `window.onfocus` and `window.onblur` 事件跟踪页面焦点

## Usage

```tsx live

function Demo() {
  const focus = useWindowsFocus();
  return (
    <div>
      <p>
        {focus
          ? "💡 点击文档外的地方来失去焦点。"
          : "ℹ 标签页失去焦点"}
      </p>
    </div>
  );
};

```
