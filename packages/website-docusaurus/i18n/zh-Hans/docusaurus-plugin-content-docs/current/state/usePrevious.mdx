# usePrevious

跟踪更新前的值

## Usage

```tsx live

function Demo() {
  const [count, setCount] = useState(0);
  const prevCount = usePrevious(count);

  return (
    <div>
      <button onClick={() => setCount(count + 1)}>+</button>
      <button onClick={() => setCount(count - 1)}>-</button>
      <p>
        现在: {count}, 之前: {prevCount}
      </p>
    </div>
  );
};

```
