# useToggle

优雅管理 `Boolean` 值。

## Usage

```tsx live

function Demo() {
  const [on, toggle] = useToggle(true);

  return (
    <div>
      <div>{on ? "开启" : "关闭"}</div>
      <button onClick={toggle}>切换</button>
      <button onClick={() => toggle(true)}>设为开启</button>
      <button onClick={() => toggle(false)}>设为关闭</button>
    </div>
  );
};
```
