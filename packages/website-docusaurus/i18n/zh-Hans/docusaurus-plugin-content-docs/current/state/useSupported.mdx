# useSupported

检测浏览器是否支持某些新的特性

## Usage

```tsx live

function Demo() {
  const isSupported = useSupported(() => "EyeDropper" in window);
  return (
    <div>
      <p>
        window.EyeDropper 在您的浏览器中是 {isSupported ? "支持" : "不支持"}
      </p>
    </div>
  );
};

```
