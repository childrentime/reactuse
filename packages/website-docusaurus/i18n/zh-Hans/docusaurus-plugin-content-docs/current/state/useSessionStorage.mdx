# useSessionStorage

轻松管理 `sessionStorage`

## Usage

```tsx live

function Demo() {
  const [value, setValue] = useSessionStorage("my-key", "key");

  return (
    <div>
      <div>Value: {value}</div>
      <button onClick={() => setValue("bar")}>bar</button>
      <button onClick={() => setValue("baz")}>baz</button>
      <button onClick={() => setValue(null)}>Remove</button>
    </div>
  );
};

```
