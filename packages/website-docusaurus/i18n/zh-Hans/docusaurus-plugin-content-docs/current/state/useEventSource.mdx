# useEventSource

`useEventSource` 是一个允许你订阅 EventSource 并实时接收更新的 hook。

```tsx live
function Demo() {
  const { status, data, error, close, open } = useEventSource(
    "https://sse.dev/test",
    [],
    {
      immediate: false,
    }
  );

  return (
    <div>
      <div>状态: {status}</div>
      <div>数据: {JSON.stringify(data)}</div>
      <div>错误: {error}</div>
      <button onClick={open}>打开</button>
      <button onClick={close}>关闭</button>
    </div>
  );
}
```
