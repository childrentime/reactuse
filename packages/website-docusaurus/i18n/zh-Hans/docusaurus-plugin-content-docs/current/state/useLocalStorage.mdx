# useLocalStorage

轻松管理 `localStorage`

## Usage

```tsx live

function Demo() {
  // bind string
  const [value, setValue] = useLocalStorage("my-key", "key");

  return (
    <div>
      <div>值: {value}</div>
      <button onClick={() => setValue("bar")}>设置为bar</button>
      <button onClick={() => setValue("baz")}>设置为baz</button>
      {/* delete data from storage */}
      <button onClick={() => setValue(null)}>移除</button>
    </div>
  );
};

```
