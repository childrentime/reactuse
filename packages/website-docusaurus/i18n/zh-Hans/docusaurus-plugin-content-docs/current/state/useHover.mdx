# useHover

跟踪鼠标是否在元素上。

## Usage

```tsx live

function Demo() {
  const ref = useRef<HTMLDivElement>(null);
  const hovered = useHover(ref);
  return <div ref={ref}> {hovered ? "是" : "否"}</div>;
};

```
