# useMobileLandscape

跟踪移动设备是否处于横屏模式。

## Usage

```tsx live

function Demo() {
  const isMobileLandscape = useMobileLandscape();
  return <p>是否是移动端横屏： {JSON.stringify(isMobileLandscape)}</p>;
};
```
