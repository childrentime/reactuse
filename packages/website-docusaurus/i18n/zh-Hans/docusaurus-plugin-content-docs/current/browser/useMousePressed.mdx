# useMousePressed

跟踪鼠标按下状态

## Usage

```tsx live

function Demo() {
  const [mouse, type] = useMousePressed();

  return (
    <div>
      <p>按下: {JSON.stringify(mouse)}</p>
      <p>来源类型: {JSON.stringify(type)}</p>
    </div>
  );
};

```
