# useTitle

设置文档标题。

## Usage

```tsx live

function Demo() {
  const [title, setTitle] = useState("title");

  useTitle(title);

  return (
    <div>
      <button onClick={() => setTitle("newTitle")}>更改标题</button>
    </div>
  );
};

```
