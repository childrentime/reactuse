# useMouse

跟踪鼠标位置。

## Usage

```tsx live

function Demo() {
  const mouse = useMouse();

  return (
    <div>
      <p>
        客户端 - x: {mouse.clientX}, y: {mouse.clientY}
      </p>
      <p>
        页面 - x: {mouse.pageX}, y: {mouse.pageY}
      </p>
      <p>
        屏幕 - x: {mouse.screenX}, y: {mouse.screenY}
      </p>
    </div>
  );
};

```
