# useDevicePixelRatio

`useDevicePixelRatio` 是一个返回屏幕的设备像素比的 hook。

## Usage

```tsx live
function Demo() {
  const { pixelRatio } = useDevicePixelRatio();

  return <p>设备像素比: {pixelRatio}</p>;
}
```
