# useIdle

跟踪页面上的用户是否处于空闲状态。

## Usage

```tsx live

function Demo() {
  const isIdle = useIdle(3e3);

  return (
    <div>
      <div>用户是否空闲: {isIdle ? "是的 😴" : "不是"}</div>
    </div>
  );
};

```
