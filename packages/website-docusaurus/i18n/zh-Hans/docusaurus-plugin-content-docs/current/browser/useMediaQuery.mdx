# useMediaQuery

轻松使用媒体查询

## Usage

```tsx live

function Demo() {
  const isWide = useMediaQuery("(min-width: 480px)");

  return <div>Screen is wide: {isWide ? "Yes" : "No"}</div>;
};

```
