# useMount

組件掛載後執行一次副作用的 React Hook

## Usage

```tsx live

function Demo() {
  const [value, setValue] = useState("未掛載");
  useMount(() => {
    setValue("已掛載");
  });
  return <div>{value}</div>;
};

```
