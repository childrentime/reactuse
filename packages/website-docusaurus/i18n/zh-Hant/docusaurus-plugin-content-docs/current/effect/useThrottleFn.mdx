# useThrottleFn

React hooks that [throttle](https://lodash.com/docs/4.17.15#throttle) function


## Usage

```tsx live

function Demo() {
  const [value, setValue] = useState(0);
  const { run } = useThrottleFn(() => {
    setValue(value + 1);
  }, 500);

  return (
    <div>
      <p style={{ marginTop: 16 }}> 點擊次數：{value} </p>
      <button type="button" onClick={run}>
        快速點擊！
      </button>
    </div>
  );
};

```
