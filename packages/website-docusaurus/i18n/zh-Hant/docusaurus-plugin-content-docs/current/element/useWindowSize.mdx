# useWindowSize

跟蹤視窗大小的 React Hook。

## Usage

```tsx live

function Demo() {
  const { width, height } = useWindowSize();

  return (
    <div>
      <p>
        寬度: {width}, 高度: {height}
      </p>
    </div>
  );
};

```
