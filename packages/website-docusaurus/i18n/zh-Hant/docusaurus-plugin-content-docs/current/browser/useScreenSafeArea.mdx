# useScreenSafeArea

跟蹤 `env(safe-area-inset-*)` 值

## Usage

```tsx live

function Demo() {
  const [top, right, bottom, left] = useScreenSafeArea();
  return (
    <div>
      <div>
        頂部：<>{top}</>
      </div>
      <div>
        底部：<>{bottom}</>
      </div>
      <div>
        左側：<>{left}</>
      </div>
      <div>
        右側：<>{right}</>
      </div>
    </div>
  );
}

```
