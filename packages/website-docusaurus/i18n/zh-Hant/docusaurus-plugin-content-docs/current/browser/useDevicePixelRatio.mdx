# useDevicePixelRatio

`useDevicePixelRatio` 是一個返回螢幕的設備像素比的 hook。

## Usage

```tsx live
function Demo() {
  const { pixelRatio } = useDevicePixelRatio();

  return <p>設備像素比：{pixelRatio}</p>;
}
```
