# usePreferredLanguages

`usePreferredLanguages` 是一個返回用戶首選語言的字符串數組的 hook。它使用 `navigator.languages`。

## Usage

```tsx live
function Demo() {
  const preferredLanguages = usePreferredLanguages();
  return <div>首選語言：{JSON.stringify(preferredLanguages)}</div>;
}
```
