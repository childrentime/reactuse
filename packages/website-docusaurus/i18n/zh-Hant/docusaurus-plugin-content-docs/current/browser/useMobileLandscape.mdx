# useMobileLandscape

跟蹤移動設備是否處於橫屏模式。

## Usage

```tsx live

function Demo() {
  const isMobileLandscape = useMobileLandscape();
  return <p>是否是移動端橫屏： {isMobileLandscape ? "是" : "否"}</p>;
};
```
