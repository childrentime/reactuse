# useMouse

跟蹤滑鼠位置。

## Usage

```tsx live

function Demo() {
  const mouse = useMouse();

  return (
    <div>
      <p>
        客戶端 - x: {mouse.clientX}, y: {mouse.clientY}
      </p>
      <p>
        頁面 - x: {mouse.pageX}, y: {mouse.pageY}
      </p>
      <p>
        螢幕 - x: {mouse.screenX}, y: {mouse.screenY}
      </p>
    </div>
  );
};

```
