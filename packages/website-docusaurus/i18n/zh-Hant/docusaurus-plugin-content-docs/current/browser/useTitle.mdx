# useTitle

設置文檔標題。

## Usage

```tsx live

function Demo() {
  const [title, setTitle] = useState("標題");

  useTitle(title);

  return (
    <div>
      <button onClick={() => setTitle("新標題")}>更改標題</button>
    </div>
  );
};

```
