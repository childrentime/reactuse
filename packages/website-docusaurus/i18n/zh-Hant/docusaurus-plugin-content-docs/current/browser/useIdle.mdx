# useIdle

跟蹤頁面上的用戶是否處於空閒狀態。

## Usage

```tsx live

function Demo() {
  const isIdle = useIdle(3e3);

  return (
    <div>
      <div>用戶是否空闒：{isIdle ? "是的 😴" : "不是"}</div>
    </div>
  );
};

```
