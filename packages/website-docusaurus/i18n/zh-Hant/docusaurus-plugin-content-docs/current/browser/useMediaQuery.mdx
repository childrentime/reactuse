# useMediaQuery

輕鬆使用媒體查詢

## Usage

```tsx live

function Demo() {
  const isWide = useMediaQuery("(min-width: 480px)");

  return <div>螢幕是否寬屏：{isWide ? "是" : "否"}</div>;
};

```
