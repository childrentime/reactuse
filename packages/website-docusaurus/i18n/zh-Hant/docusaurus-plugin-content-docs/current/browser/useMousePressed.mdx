# useMousePressed

跟蹤滑鼠按下狀態

## Usage

```tsx live

function Demo() {
  const [mouse, type] = useMousePressed();

  return (
    <div>
      <p>按下狀態：{JSON.stringify(mouse)}</p>
      <p>來源類型：{JSON.stringify(type)}</p>
    </div>
  );
};

```
