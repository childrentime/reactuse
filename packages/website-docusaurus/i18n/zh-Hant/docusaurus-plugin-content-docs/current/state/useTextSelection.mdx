# useTextSelection

跟蹤用户选中的文字

## Usage

```tsx live noInline

function Demo() {
  const selection = useTextSelection();

  return (
    <div style={{ padding: 40 }}>
      <p>
        在這裡或頁面上的任何地方選擇一些文本，它將顯示在下面
      </p>

      <div>選中的文本: {selection?.toString()}</div>
    </div>
  );
};

render(<Demo/>)
```
