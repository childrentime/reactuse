# useEventSource

`useEventSource` 是一個允許你訂閱 EventSource 並實時接收更新的 hook。

```tsx live
function Demo() {
  const { status, data, error, close, open } = useEventSource(
    "https://sse.dev/test",
    [],
    {
      immediate: false,
    }
  );

  return (
    <div>
      <div>狀態: {status}</div>
      <div>數據: {JSON.stringify(data)}</div>
      <div>錯誤: {error}</div>
      <button onClick={open}>開啟</button>
      <button onClick={close}>關閉</button>
    </div>
  );
}
```
