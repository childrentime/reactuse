# useFirstMountState

返回組件是否剛剛掛載的 React Hook

## Usage

```tsx live

function Demo() {
  const isFirstMount = useFirstMountState();
  const [render, reRender] = useState(0);

  return (
    <div>
      <span>這個組件剛剛掛載: {isFirstMount ? "是" : "否"}</span>
      <br />
      <button onClick={() => reRender(1)}>{render}</button>
    </div>
  );
};

```
