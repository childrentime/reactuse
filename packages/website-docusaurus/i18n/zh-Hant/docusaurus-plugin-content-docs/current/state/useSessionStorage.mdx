# useSessionStorage

輕鬆管理 `sessionStorage`

## Usage

```tsx live

function Demo() {
  const [value, setValue] = useSessionStorage("my-key", "key");

  return (
    <div>
      <div>值: {value}</div>
      <button onClick={() => setValue("bar")}>設置為bar</button>
      <button onClick={() => setValue("baz")}>設置為baz</button>
      <button onClick={() => setValue(null)}>移除</button>
    </div>
  );
};

```
