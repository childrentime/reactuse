# useToggle

優雅管理 `Boolean` 值。

## Usage

```tsx live

function Demo() {
  const [on, toggle] = useToggle(true);

  return (
    <div>
      <div>{on ? "ON" : "OFF"}</div>
      <button onClick={toggle}>切換</button>
      <button onClick={() => toggle(true)}>設置為開啟</button>
      <button onClick={() => toggle(false)}>設置為關閉</button>
    </div>
  );
};
```
