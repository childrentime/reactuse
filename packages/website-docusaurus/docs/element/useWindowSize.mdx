# useWindowSize

React Element Hooks that tracks window size


## Usage

```tsx live
function Demo() {
  const { width, height } = useWindowSize();

  return (
    <div>
      <p>
        width: {width}, height: {height}
      </p>
    </div>
  );
};

```
