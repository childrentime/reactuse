# useMobileLandscape

React hook that returns `true` if the mobile device is in landscape mode and `false` otherwise.

## Usage

```tsx live
function Demo() {
  const isMobileLandscape = useMobileLandscape();
  return <p>isMobileLandscape: {JSON.stringify(isMobileLandscape)}</p>;
};

```
