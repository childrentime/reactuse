# useDevicePixelRatio

`useDevicePixelRatio` is a hook that returns the device pixel ratio of the screen.

## Usage

```tsx live
function Demo() {
  const { pixelRatio } = useDevicePixelRatio();

  return <p>Device pixel ratio: {pixelRatio}</p>;
};

```
