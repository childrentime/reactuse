# useMount

React lifecycle hook that executes a function after the component is mounted


## Usage

```tsx live
function Demo() {
  const [value, setValue] = useState("UnMounted");
  useMount(() => {
    setValue("Mounted");
  });
  return <div>{value}</div>;
};

```
